# ☁️ 50 Days of Azure Challenge – KodeKloud

Welcome to my **50 Days of Azure Challenge** powered by KodeKloud 🚀  

This repository contains my daily hands-on Azure cloud tasks, solutions, notes, and learning journey as I explore Azure services, DevOps practices, automation, security, infrastructure, and cloud engineering concepts.



# 📌 About the Challenge

The goal of this challenge is to:

- Build consistency in cloud learning
- Gain practical Azure experience
- Improve DevOps and Infrastructure skills
- Work with real-world cloud scenarios
- Practice automation and troubleshooting
- Document the learning journey publicly

Each day includes:
- Problem Statement
- Step-by-step Solution
- Commands Used
- Key Learnings
- Challenges Faced
